### F.1 工具自检功能

针对我在开发 MCP Agent 时遇到的问题，我开发工具自检功能，这个功能可以在你的 MCP 开发初期快速帮你完成所有 tool 的执行验证，从而大幅度降低交互测试中对 mcp 试错的成本，加速 MCP 的开发和上线。

![](https://linux.do/uploads/default/optimized/4X/6/8/d/68d8fbdc71262dcbb1856ed1f9a4d76c2013bf71_2_1380x820.jpeg)

该功能提供了如下的一些具体功能：

1.  提供预设拓扑
2.  允许用户在受限范围内高度定制测试顺序
3.  对于结果可以 pin 住，对结果进行一定的统计和可视化
4.  部分功能会在右下角出现提示

### F.2 更新面板

从 0.1.9 开始，每次更新插件后，首次启动后都会弹出一个更新面板，用于提醒大家我们这次更新的具体内容，上面也张贴了有关 OpenMCP 丰富的使用教程和学习资料。
